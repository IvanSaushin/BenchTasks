Основные сортировки это:

1. Сортировка выбором   (Selection Sort)    O(n^2)
2. Пузырьковая          (Bubble sort)       O(n^2)
3. Сортировка вставками (Insertion Sort)    O(n^2)
4. Быстрая              (Quick sort)        O(n log n) - pivot, и границы L, R.
        числа, меньше опорного - нелево. больше - направо.
5. Слиянием             (Merge sort)        O(n log n)
